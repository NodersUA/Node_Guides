# Useful Commands

_**Useful Commands**_

```bash
# Restart docker
docker restart shardeum-dashboard
```

To exit the shell, use the command

```bash
exit
```

Stop the node

```bash
operator-cli stop
```

Check node status

```bash
operator-cli gui status
```

Change the port through the terminal if necessary

```bash
operator-cli gui set port your_port_here
```
